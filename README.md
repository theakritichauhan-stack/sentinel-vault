                                                 🛡️ SENTINEL VAULT

AI-Powered Banking & Financial Advisory System


<img width="1983" height="793" alt="ChatGPT Image Jun 30, 2026 at 01_06_08 PM" src="https://github.com/user-attachments/assets/33838345-cad3-400c-b8df-82c3247e0a46" />


.Sentinel Vault is a full-stack banking application built using Spring Boot that combines secure digital banking with an AI-powered financial advisor.

.The application enables users to perform banking operations such as deposits, withdrawals, fund transfers, transaction history management, and PDF statement generation, while also providing AI-driven financial guidance to help users make better financial decisions.

.Designed using a layered architecture, the project demonstrates clean code practices, authentication, database management, and responsive user interface development.

| Withdraw Money     | Budget Suggestions   | Session Authentication     | Transaction History |
| Fund Transfer      | Savings Guidance     | Form Validation            | Dashboard           |
| Account Management | Financial Awareness  | Exception Handling         | Profile Management  |
